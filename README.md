# Air Quality Index Prediction Project

This project predicts Air Quality Index (AQI) using machine learning models.
It is developed as part of the Infosys project work.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost, LightGBM
- Random Forest
- Joblib

## Project Features
- Data cleaning and preprocessing
- Handling missing values
- Model training using ensemble techniques
- AQI prediction with high accuracy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the training script or notebook
